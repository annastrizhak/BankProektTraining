# MyAwesomeProject

Это проект для выполнения домашних заданий по курсу Python-разработки.

## Описание

Проект содержит следующие модули:
`masks` c функциями для получения маски для счета и карт;
`widget` с функциями для замаскированных данных и преобразования даты;
`processing` с функциями для фильтрации и сортировки данных.

### Тестирование

Проект использует `pytest` для выполнения юнит-тестов и `pytest-cov` для анализа покрытия кода тестами.

#### Запуск тестов

Для запуска всех тестов проекта выполните следующую команду в корневой директории:

```bash
pytest
